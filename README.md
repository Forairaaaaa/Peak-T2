#### 硬件配置：

| 主控     | ESP32-PICO-D4                                             |
| -------- | --------------------------------------------------------- |
| 屏幕     | ST7789 IPS 2.0英寸LCD  SPI接口   分辨率：240x320          |
| 输入设备 | CST026 2.0英寸电容触摸屏  分辨率：240x320  、  旋转编码器 |
| 输出设备 | 一颗用户LED、蜂鸣器                                       |
| 储存     | TF卡槽  SPI接口                                           |
| IMU      | MPU6050  I2C接口                                          |
| GPS      | ATGM336H模块  UART接口                                    |
| 电池     | Li-ion 3.7V 603450 1200mAh                                |
| 电源管理 | ME6215C33M5G + MCP73831                                   |
| USB接口  | Type-C座，CP2102自动下载电路                              |
